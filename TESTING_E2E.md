# VertiAI — คู่มือทดสอบ End-to-End (Session 6)

เอกสารนี้ใช้ทดสอบบนเครื่องผู้ใช้จริง เพราะต้องดึงข้อมูลสดจาก University of Wyoming
ซึ่งถูกบล็อกในสภาพแวดล้อม sandbox (HTTP 403) — การทดสอบ live จึงต้องทำที่นี่

## เตรียมการ

```bash
cd vertiai
python -m venv .venv && source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt
flask --app app run --debug
```

เปิด `http://127.0.0.1:5000/` ค้างหน้าต่าง terminal ไว้ดู log เวลาตอบสนอง

## 1. ทดสอบ 2 สถานีจริง (โหมด live ดึง Wyoming)

| สถานี | รหัส | ขั้นตอน | ผลที่คาดหวัง |
|-------|------|---------|--------------|
| กรุงเทพฯ (บางนา) | 48453 | เลือกสถานี + วันที่เมื่อวาน + 00Z → กด **วิเคราะห์** | Skew-T, Hodograph, KPI, AI score ขึ้นครบ |
| เชียงใหม่ | 48327 | เลือกสถานี + วันที่เมื่อวาน + 00Z → กด **วิเคราะห์** | แสดงผลครบเช่นกัน |

ตรวจสอบเพิ่มเติมแต่ละสถานี:
- [ ] เวลาตอบครั้งแรก (live) แสดงใน status bar; ดู log `(/api/sounding ... live ...ms)` — ครั้งแรกอาจ > 2s ได้
- [ ] กด **วิเคราะห์** ซ้ำชุดเดิม → ต้องตอบจากแคช `(cached)` และ < 2s (log แสดง `cache`)
- [ ] ชั้นผกผัน (แถบส้ม) และจุด LCL/LFC/EL ปรากฏถูกตำแหน่ง
- [ ] เลือก **พื้นที่** (เช่น นครราชสีมา) แล้ววิเคราะห์ → ส่วนเกณฑ์เฉพาะถิ่นเปลี่ยนตาม

## 2. ทดสอบ fallback เมื่อ Wyoming ล่ม

- [ ] เลือกวันที่อนาคต/วันที่ไม่มีข้อมูล → ต้องเห็น **แถบ error สีแดง** พร้อมปุ่ม *ลองใหม่* และ *อัปโหลดไฟล์แทน*
- [ ] ปิดเน็ต แล้วกดวิเคราะห์ → ต้องเห็นข้อความ "เชื่อมต่อเซิร์ฟเวอร์ไม่ได้"

## 3. ทดสอบอัปโหลดไฟล์ (1 ไฟล์)

ที่หน้า `/data`:
- [ ] บันทึกหน้า text-list ของ Wyoming เป็นไฟล์ `.txt`/`.html` แล้วอัปโหลด → ตารางสรุปดัชนีขึ้น
- [ ] ทดสอบวางข้อความ text-list ลงช่อง textarea → ได้ผลเช่นกัน
- [ ] อัปโหลดไฟล์ผิดรูปแบบ (เช่น .txt เปล่า) → ต้องเห็นข้อความ error ไม่ใช่หน้าเปล่า

## 4. ทดสอบทุก Role

ที่หน้า `/settings` เลือกบทบาทแล้วบันทึก จากนั้นกลับไปแดชบอร์ดวิเคราะห์:

| Role | KPI ที่ต้องถูกไฮไลต์ (ขอบเน้น) |
|------|-------------------------------|
| นักบิน (pilot) | Freezing 0°C, LCL, LFC, EL |
| นักอุตุนิยมวิทยา (meteorologist) | CAPE, CIN, LI, K-Index, PWAT |
| นักวิจัย (researcher) | ทุกตัว |
| เกษตรกร (farmer) | CAPE, PWAT, K-Index |

- [ ] ป้ายบทบาท (role-badge) ปรากฏบน topbar
- [ ] สลับภาษา TH/EN → ป้ายและ KPI เปลี่ยนภาษาถูกต้อง
